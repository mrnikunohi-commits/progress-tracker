📱 Instagram Graph API

セットアップ手順:
1. Instagram アカウントを「プロアカウント（クリエイター/ビジネス）」に切替
2. Facebook ページとリンク（プロフィール編集 → ページのリンク）
3. Meta for Developers でアプリ作成（用途: その他/ビジネス）
4. Graph API Explorer で instagram_basic, instagram_manage_insights, pages_show_list, pages_read_engagement 権限のトークン発行
5. Access Token Debugger で「Extend Token」→ 長期トークン（60日）に変換
6. /me/accounts → {page-id}?fields=instagram_business_account で IG User ID（数字）を取得
※ トークンはこのブラウザ内に保存されます。共有リンク先のクラウドにも同期されるので、共有相手とのみ共有してください。

⏱️ 勤怠アプリ 自作API

自作の勤怠アプリのエンドポイントを登録してください。
GETリクエストで JSON を返し、{"days8h": 18} のようなキー指定で日数を読み取ります。
認証ヘッダは Header-Name: value 形式で1行（任意）。
※ そのアプリが CORS を許可している必要があります（Access-Control-Allow-Origin: * など）。
